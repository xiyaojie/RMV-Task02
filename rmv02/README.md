## 一个问题
在输出红色轮廓的时候 发现输出了很多0值 询问gpt以后发现是由于 findContours() 函数可能会找到非常小的轮廓，甚至噪声，导致输出 0 面积的红色区域。
于是设置了一个面积阈值来滤掉噪声
